## 📋 Overview

AgroStar is a comprehensive agricultural e-commerce website designed for farmers to access high-quality agricultural products, farming tips, and expert guidance. The website features a modern, responsive design with support for **3 languages** (Marathi, Hindi, English) to serve the diverse farming community.

## ✨ Features

### 🌐 Multi-Language Support
- **Marathi** (मराठी) - Primary language for Maharashtra farmers
- **Hindi** (हिंदी) - National language for wider reach
- **English** - International language for broader audience
- Language preference saved in localStorage

### 📱 Responsive Design
- Fully responsive layout for all devices
- Mobile-first approach
- Smooth animations and hover effects
- Optimized for all screen sizes

### 🛒 Product Categories
- **Seeds** - 22+ premium quality seeds (vegetables, cereals, pulses, oilseeds, fruits)
- **Organic Fertilizers** - 22+ natural fertilizers (vermicompost, manure, liquid fertilizers)
- **Chemical Fertilizers** - 22+ NPK and micronutrient fertilizers
- **Pesticides** - 22+ insecticides, fungicides, bio-pesticides
- **Agro Tonics** - 22+ growth promoters and yield boosters
- **Farming Equipment** - 22+ modern farming tools and machinery

### 📚 Additional Features
- **Farming Tips** - 22+ expert tips with category filters
- **About Us** - Company story, mission, vision, team, timeline
- **Contact Page** - Working contact form, Google Maps, WhatsApp integration
- **Product Filters** - Category-wise product filtering
- **Product Badges** - Hot, Bestseller, New, Organic labels
- **Modal Popups** - Detailed product/tip information
- **Local Storage** - Contact messages stored locally
